# 📝 在线 Markdown 编辑器

一个简洁优雅的在线 Markdown 编辑器，无需安装，打开即用。支持实时预览、本地自动保存、快捷工具栏、性能优化等功能。

🌐 **在线预览**: [markdown.dooo.fun](https://markdown.dooo.fun)

![编辑器预览](https://img.shields.io/badge/在线体验-markdown.dooo.fun-blue?style=flat-square)

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ⚡ **实时预览** | 左侧编辑，右侧即时渲染 Markdown，支持防抖优化 |
| 💾 **自动保存** | 内容自动保存到浏览器本地存储，下次自动恢复 |
| 🛠️ **快捷工具栏** | 一键插入常用 Markdown 语法，支持 15+ 种快捷操作 |
| 📂 **导入/导出** | 支持 `.md`、`.markdown`、`.txt` 文件导入导出 |
| 🎨 **代码高亮** | Atom One Dark 主题，支持多种编程语言 |
| ⌨️ **快捷键** | `Ctrl+S` / `Cmd+S` 快速保存，支持多种快捷键 |
| 📱 **响应式设计** | 完美适配桌面端和移动端 |
| 🚀 **性能优化** | 防抖节流处理，大文本不卡顿 |

## 🚀 快速开始

### 在线使用

直接在浏览器中打开 `index.html` 文件即可使用：

```bash
# 方式一：直接打开文件
双击 index.html

# 方式二：使用本地服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 项目结构

```
online-md/
├── index.html    # 主文件（包含所有代码）
└── README.md     # 说明文档
```

## 📖 使用指南

### 工具栏功能

编辑器提供便捷的 Markdown 工具栏，支持以下快捷操作：

| 分组 | 按钮 | 功能 | 语法 |
|------|------|------|------|
| 标题 | H1 H2 H3 | 插入标题 | `# 标题` `## 标题` `### 标题` |
| 格式 | 𝐁 𝐼 ̶S̶ | 文本格式 | `**粗体**` `*斜体*` `~~删除线~~` |
| 媒体 | 🔗 🖼️ | 链接与图片 | `[文本](url)` `![描述](url)` |
| 代码 | ‹› 📄 | 代码插入 | `` `代码` `` ` ```代码块``` ` |
| 列表 | • 1. ☑️ | 列表插入 | `- 项目` `1. 项目` `- [ ] 任务` |
| 其他 | ❝ ▦ — | 引用表格分隔线 | `> 引用` `\|表格\|` `---` |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` / `Cmd + S` | 保存内容到本地 |

### 本地缓存

- 编辑内容会自动保存到浏览器本地存储
- 下次打开页面自动恢复上次编辑的内容
- 底部状态栏显示保存状态（✓ 已保存 / 💾 保存中）

## 🛠️ 技术栈

- **Vue 3** (CDN) - 响应式框架
- **Marked.js** - Markdown 解析器
- **Highlight.js** - 代码语法高亮 (Atom One Dark 主题)
- **GitHub Markdown CSS** - Markdown 基础样式

所有依赖均通过 CDN 引入，无需安装任何依赖。

## ⚡ 性能优化

- **防抖渲染** - Markdown 渲染 300ms 防抖，避免频繁渲染
- **节流统计** - 字符/行数/词数统计 200ms 节流
- **长文本处理** - 超过 50,000 字符时智能截断，防止卡顿
- **代码高亮限制** - 限制单次高亮代码块数量和长度
- **非阻塞渲染** - 使用 requestIdleCallback 避免阻塞主线程
